第三模組　標準條款

第一條　工程範圍與內容

乙方承攬甲方委託之室內裝修工程（以下簡稱「本工程」），工程範圍以雙方簽認之工程報價單（附件一）及設計圖說（附件二）為準。

工程範圍如有疑義，以設計圖說為優先，設計圖說未載明者，以工程報價單為準；兩者均未載明者，以書面協議補充之。

乙方應依建築技術規則、內政部頒訂相關法規及台灣現行國家標準（CNS）施作，並取得依法應辦理之相關許可。

第二條　工期與完工期限

乙方應於開工日起算之約定施工日曆天內完成本工程，並辦理驗收交付。

下列情形發生時，經甲方書面確認，得合理展延工期，並不視為乙方違約：

  1. 因第十一條所定不可抗力事由；
  2. 因甲方指示設計變更或追加工程；
  3. 因甲方未依約支付工程款，致乙方依法停工；
  4. 政府機關命令停工或其他非可歸責於乙方之情事。

工期展延申請，乙方應於事由發生後 7 個工作日內以書面通知甲方，逾期不得主張展延。

第三條　工程價款與付款方式

本工程總價依第二模組專案條款付款分期計畫表所訂金額為準，付款方式採「階段驗收付款機制」。

  1. 驗收付款原則：每期款項以甲方對應階段工程驗收簽認後，應於 7 個工作日內完成匯款或現金給付，甲方不得無故拒絕驗收。
  2. 驗收爭議：如甲方對工程品質有疑義，應以書面載明具體瑕疵項目通知乙方；乙方應於收到通知後 7 個工作日內完成修繕，修繕完成後甲方應即辦理複驗。
  3. 付款遲延：甲方逾期未付工程款者，每逾一日按遲付金額萬分之三計算遲延利息；逾期超過 14 日者，乙方得停工並書面通知甲方，停工期間工期相應展延，相關損失由甲方負擔。
  4. 發票開立：乙方應依《加值型及非加值型營業稅法》規定開立統一發票；如乙方為免稅銷售者，應依法開立收據。

第四條　設計變更與追加減帳

本工程施工期間，如甲方提出設計變更或追加減帳事項，需依下列程序辦理，否則不生效力：

  1. 甲方以書面提出變更需求；
  2. 乙方於 5 個工作日內提供書面變更估價單，載明工程項目、費用增減及工期影響；
  3. 雙方簽署「工程變更確認單」後，方得施工；
  4. 口頭指示概不列為有效變更依據。

追加工程款應依前項確認單所訂金額計算，並納入付款分期計畫中；減帳工程款則於最終驗收時統一結算。

第五條　材料與施工品質

乙方使用之材料應符合附件三（材料設備表）所載規格，且均需符合台灣現行 CNS 國家標準、環保法規及建築技術規則之相關規定。

甲方得隨時進場查驗材料規格及施工品質。如發現材料不符合約規範，乙方應於甲方書面通知後 3 個工作日內更換；逾期不換者，甲方得扣除對應期款，或自行採購並向乙方求償差價。

本工程使用之油漆、黏著劑、板材等建材，其甲醛含量、揮發性有機化合物（VOC）排放量不得超過《室內空氣品質管理法》所定相關標準。

第六條　驗收與交付

本工程採分階段驗收，驗收節點依第二模組付款分期計畫表所列條件執行，每次驗收雙方均應簽署「階段驗收紀錄表」。

竣工驗收程序：

  1. 乙方完工後以書面通知甲方，甲方應於收到通知後 5 個工作日內進行最終驗收；
  2. 驗收合格，雙方簽署「竣工驗收證明書」，乙方憑此請領尾款；
  3. 驗收發現缺失，乙方應於約定期限內完成改善，改善後進行複驗；
  4. 甲方逾期不辦理驗收，視為驗收通過，乙方得請領尾款。

⚠ 風險控管強化條款

第七條　保固條款 重要

乙方應自竣工驗收通過之日起，對本工程提供至少一（1）年之保固服務，保固期間因施工瑕疵或材料品質不符所致之損壞，乙方應無償修繕。

保固範圍包含：木作工程、泥作工程、水電配管配線、油漆粉刷、輕隔間、天花板、地板鋪設及固定式家具等。

下列情形不在保固範圍：

  1. 因甲方或第三人使用不當、人為破壞所致之損壞；
  2. 因不可抗力事由（地震、颱風等天災）所致之損壞；
  3. 甲方自行改裝或委由第三方施作所造成之損壞；
  4. 正常磨損及自然老化。

保固期滿後，乙方仍得依有償方式提供維修服務。

第八條　違約責任 關鍵條款

（一）乙方違約責任：

  1. 工期延遲：乙方無正當理由未能於約定期限完工者，每逾一日，應按工程總價之千分之一（‰1）計算違約金，惟違約金總額以不超過工程總價之 10% 為上限；超過 30 日者，甲方得終止合約並請求損害賠償。
  2. 施工瑕疵未改善：乙方於保固期內未依時修繕者，甲方得委由第三方施作，費用由乙方負擔，並得向乙方主張費用加計 20% 之管理費。
  3. 擅自停工：乙方非因可歸責於甲方之事由擅自停工逾 7 日者，視為乙方違約，甲方得終止合約並請求損害賠償。

（二）甲方違約責任：

  1. 付款遲延：依第三條第三款計算遲延利息；甲方逾期付款超過 30 日者，乙方得終止合約並請求應付款項及損害賠償。
  2. 無正當理由終止：甲方如非因可歸責於乙方之事由而終止合約，應賠償乙方已完成工程費用，並支付工程總價 10% 之違約金。

雙方同意上述違約金為損害賠償總額預定性質，法院得依情事酌減之。

第九條　預付款與履約保障 保護甲方

本合約採下列履約保障措施，以平衡雙方風險：

  1. 訂金上限：簽約時之第一期款（訂金）不得超過工程總價之 30%；如有超過，視為無效超額部分，甲方得拒絕給付。
  2. 分段付款保護：付款以「階段驗收通過」為前提，未驗收通過前甲方無給付下一期款之義務。
  3. 履約保障（擇一）：
    • 乙方提供銀行或保險公司開立之「履約保證書」；
    • 各期款設第三方信託帳戶，依驗收結果分批撥付；
    • 免附加履約保障（雙方合意）。
  4. 材料預付款：甲方如預付特殊訂製材料費，乙方應提供相應訂購單據（賣方單據）為憑。

第十條　爭議處理

本合約適用中華民國法律。雙方如有爭議，應先本於誠信原則以協商方式解決；協商不成，得依下列方式處理：

  1. 調解：得向工程所在地之鄉鎮市調解委員會或消費者保護機關申請調解；
  2. 訴訟：以工程所在地之地方法院為第一審管轄法院。

第十一條　不可抗力

下列情形發生致使一方無法履行義務者，不構成違約，但應於事由發生後 3 日內書面通知對方：

  1. 颱風、地震、洪水、土石流等天然災害；
  2. 戰爭、暴動、政府機關命令（如封城、禁建令）；
  3. 全國性罷工、重大傳染病疫情（依行政院公告為準）；
  4. 其他雙方均無法合理預見及控制之事由。

不可抗力情形消滅後，雙方應立即恢復履約，並協商調整工期及相關費用。

第十二條　其他約定

  1. 合約完整性：本合約連同附件構成雙方完整合意，其效力優於雙方先前任何口頭或書面協議。
  2. 合約修改：任何修改均需雙方書面簽認，方生效力。
  3. 可分割性：本合約任一條款被認定無效，不影響其餘條款之效力。
  4. 個人資料保護：雙方同意依《個人資料保護法》規定，僅將本合約所載個人資料用於合約履行目的，不得洩漏或作其他用途。
  5. 合約份數：本合約一式兩份，甲乙方各執一份為憑。
